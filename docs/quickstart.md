# 처음 5분 — 실제 결정 하나로 끝까지 돌려보기

설치 끝났다고 가정. ([install_guide.md](install_guide.md) 참고)

## 시나리오: "친구 회사가 같이 일하자고 제안함"

이런 상황이라고 치자.

> 친구 A가 자기 스타트업에 같이 합류하자고 함. 지분 1.5%, 연봉 현재보다 -20%. 1주일 고민 끝에 **거절**하기로 함. 이유는: (1) 친구와 일하면서 관계 깨질 위험 (2) 현 회사 프로젝트 6개월만 더 하면 마무리 (3) 지분 1.5%는 액수보다 너무 적게 느껴짐.

이걸 기록해보자.

## Step 1 — Claude Code에서 결정을 말로 적어

평소 Claude한테 고민 털어놓듯이 적되, *결론*이 나면 이 형식으로 한 줄 박아:

```
결정 / decision: 친구회사제안 | A 친구 스타트업 합류 거절, 관계 리스크 + 현직 마무리 우선
```

(영어로 써도 됨 — `decision: friend-startup-offer | Declined A's startup offer ...` 동일)

토픽은 `친구회사제안` 같은 짧은 슬러그. 내용은 한 줄로 "뭘 결정했고 왜인지" 핵심만.

추가로 더 박을 수 있어:

```
판단 / analysis: 친구회사제안 | 지분 1.5%는 액수 자체가 의미 없는 수준, 시장 평균 이하
원칙 / principle: 관계 | 친한 사람과 같이 일할 때는 지분/조건이 시장 + 20% 이상이어야 위험 보상
```

세 종류는:
- **결정 / decision** — *이번 한 번의 선택*
- **판단 / analysis** — *과정에서 발견한 사실*
- **원칙 / principle** — *앞으로의 룰*

## Step 2 — Claude한테 박아달라고 해

같은 대화 안에서 그냥 이렇게 말해:

> "방금 적은 마커들 trace에 기록해줘"

또는

> "decision-extractor 돌려"

Claude가 `state/recall_trace.jsonl`에 한 줄씩 추가하고 결과를 알려줌.

### 또는: Claude가 자동으로 제안 (auto-suggest)

평소처럼 대화하다가 결정이 나오면 Claude가 알아서 물어봐:

> 📓 Want me to record this as a `decision` marker?
> `decision: 친구회사제안 | A 친구 스타트업 합류 거절, 관계 리스크 + 현직 마무리 우선`
> (`y` = save / `n` = skip / `e` = edit)

`y` 한 번 누르면 끝. 형식 외울 필요 없음. 매번 묻는 게 부담스러우면 `n` 하면 다시 안 물음.

## Step 3 — 바로 확인

```
/recall
```

방금 박은 3줄이 시간 순으로 나옴. 끝.

## 3개월 뒤 시나리오

3개월 후, 다른 친구 B가 비슷한 제안을 함. *지난번에 어떻게 판단했더라?* 그때:

```
/recall 친구회사
```

→ Step 1에서 박은 3줄이 그대로 떠올라. "관계 리스크 / 지분 너무 적음 / 시장 +20% 룰" — 이 reasoning을 처음부터 다시 짤 필요 없음. 새 제안의 조건만 그 위에 비교하면 됨.

이게 이 도구의 전부야.

## "그냥 노션에 적으면 안 돼?"

돼. 다만 노션 페이지는 시간 지나면 *찾는 데 시간 걸림*. 이 도구는:

1. **대화 흐름 중에** 박을 수 있음 (별도 탭 열 필요 X)
2. **`/recall <키워드>`** 한 줄로 즉시 검색됨
3. **로컬 파일**이라 어디서 일하든 따라옴

노션에 같이 적어도 됨 — 이 도구는 *대화 중 즉시 박기 + 즉시 회상* 용도.

## 처음 1주일 — 이렇게 써봐

매일 대화 끝에 *오늘 결정한 게 1개라도 있으면* 1줄 박기. 1주일이면 5~10줄 쌓임. 그 다음 주에 `/recall` 한번 쳐봐 — *내가 일주일 전에 뭘 결정했지?* 가 한 화면에 뜸. 이게 작동하는 순간임.

## 안 박아야 하는 것 (중요)

이 도구의 가치는 *signal*에서 옴, volume에서 X.

- ❌ "오늘 회의함" — 회의록 X
- ❌ "PR 머지함" — 코드 히스토리에 있음
- ❌ "기분이 안 좋음" — 일기 X
- ❌ "X가 뭔지 배움" — 학습 노트 X

✅ "A 거절 B 선택 — 왜냐면..." 같은 *진짜 선택*만.

3개월 뒤 너가 *다시 보고 싶은 줄*만 박아.
